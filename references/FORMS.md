# Form Templates by Business Type

## Restaurant Reservation

### Required Fields
- **Date**: Calendar picker or date input
- **Time**: Time slot selection or dropdown
- **Party size**: Number (typically 1-20)
- **First name**: Text input
- **Last name**: Text input (sometimes combined with first)
- **Phone**: Phone number input
- **Email**: Email input

### Optional Fields
- **Special requests**: Textarea (seating preference, allergies, celebrations)
- **Dining area**: Dropdown (indoor, outdoor, bar, private room)
- **Occasion**: Dropdown (birthday, anniversary, business)

### Confirmation Button Text
- "Complete reservation"
- "Book now"
- "Reserve"
- "Confirm reservation"

---

## Salon / Spa Appointment

### Required Fields
- **Service type**: Dropdown or list (haircut, color, manicure, massage)
- **Staff member**: Dropdown (optional, "No preference" default)
- **Date**: Calendar picker
- **Time**: Available slot selection
- **First name**: Text input
- **Last name**: Text input
- **Phone**: Phone number
- **Email**: Email input

### Optional Fields
- **Notes**: Textarea (specific requests)
- **New client**: Checkbox (first visit)

### Confirmation Button Text
- "Book appointment"
- "Confirm booking"
- "Schedule"

---

## Fitness Class

### Required Fields
- **Class type**: Selection from schedule (yoga, spinning, HIIT)
- **Date/Time**: Combined in class schedule grid
- **Instructor**: Sometimes selectable
- **First name**: Text input
- **Last name**: Text input
- **Email**: Email input
- **Phone**: Phone number

### Optional Fields
- **Equipment rental**: Checkbox
- **Experience level**: Dropdown (beginner, intermediate, advanced)
- **Waiver**: Checkbox (accept terms)

### Confirmation Button Text
- "Book class"
- "Reserve spot"
- "Sign up"

---

## Healthcare Appointment

### Required Fields
- **Provider/Doctor**: Selection
- **Visit type**: Dropdown (new patient, follow-up, consultation)
- **Date**: Calendar picker
- **Time**: Available slot selection
- **First name**: Text input
- **Last name**: Text input
- **Date of birth**: Date input
- **Phone**: Phone number
- **Email**: Email input
- **Insurance**: Provider and ID (often required)

### Optional Fields
- **Reason for visit**: Textarea
- **Preferred language**: Dropdown
- **New patient forms**: Link to complete before visit

### Confirmation Button Text
- "Schedule appointment"
- "Book visit"
- "Confirm"
